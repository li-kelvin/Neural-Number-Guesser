# Number Guesser Neural Network

This is a game developed to predict the digit drawn on screen by the user. It utilizes the MINST digit data set and uses basic neural network model.

- Python 3.6 or below
- Pygame
- Tensorflow
- Keras
- Numpy
- Matplotlib
